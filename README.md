# ECOCO FAQ Assistant

ECOCO 內部常見問題集管理系統，提供簡易的本地端介面，用於管理和編輯客服常用問答 (FAQ)。

## 功能
- 新增、編輯、刪除常見問題。
- 從本地端自動連線本機伺服器 (`http://127.0.0.1:7777`)。
- SQLite 本地資料庫，資料自動保存於 `ecoco_faq.db`。

## 執行需求
- Python
- 套件：`fastapi`, `uvicorn`

## 如何啟動
只需透過資料夾中的 `啟動ECOCO知識庫.bat` 執行即可：
1. 自動安裝所需的 Python 套件。
2. 背景啟動知識庫本地伺服器 (`faq_server.py`)。
3. 等待幾秒後將自動為您開啟網頁管理介面進行操作。
